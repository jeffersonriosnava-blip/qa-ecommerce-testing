# Bug Report - Cart Page

## ID
BUG-001

## Title
Product duplicates in cart after refreshing page

## Severity
Medium

## Priority
High

## Environment
Chrome Version 135
Windows 11

## Steps to Reproduce
1. Login into the application
2. Add a product to cart
3. Refresh the cart page

## Expected Result
The product should appear only once.

## Actual Result
The product duplicates after refreshing.
