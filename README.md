# QA E-commerce Testing Project

## Project Overview
Manual QA testing project executed on DemoBlaze e-commerce website.

## Scope
- Login testing
- Cart testing
- Purchase flow
- Navigation testing
- Form validations

## Testing Types
- Functional Testing
- UI Testing
- Negative Testing
- Smoke Testing

## Deliverables
- Test Cases
- Bug Reports
- Test Execution Summary
- Screenshots Evidence

## Tools Used
- Excel
- GitHub
- Chrome DevTools
- Jira (simulated)

## Test Environment
- Browser: Google Chrome
- OS: Windows 11
